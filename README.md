# Animated Website

A modern, animated landing page with smooth transitions and effects.

## Features

- **Logo & Menu**: Logo and menu bar slide in from the top with green styling and dynamic hover effects
- **Background Images**: Four images that slide in from the left and exit to the right every 2 seconds
- **Text Animation**: Smooth fade-in effect for the main heading
- **Buttons**: Two dynamic action buttons that fall from the top with green, white, and black color schemes

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `script.js` - JavaScript for dynamic background slideshow
- `logo.png` - Logo image
- `1.png.jpg`, `2.png.jpg`, `3.png.jpg`, `4.png.jpg` - Background images

## Usage

Open `index.html` in a web browser to view the animated website.

## Customization

- Change colors in `style.css`
- Replace background images by updating image files
- Adjust animation timing by changing interval in `script.js` (currently 2 seconds)
- Modify button styles and hover effects in `style.css`
